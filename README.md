# 手册
```
go-ethereum 一共有以下八个子模块

1.ethclient - 以太坊客户端
2.accounts  - 账户管理
3.core      - 底层区块链数据
4.abi       - 智能合约交互
5.common    - 通用工具
6.crypto    - 加密工具
7.rpc       - RPC 客户端
8.event     - 事件订阅
```
